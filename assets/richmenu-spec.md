# Rich Menu Image Spec

Prepare image for LINE Rich Menu with these specs:

- Size: `2500 x 843` pixels
- Format: `PNG` or `JPG`
- 3 horizontal zones (left/middle/right)
  - Left: `สรุปวันนี้`
  - Middle: `สรุปเดือนนี้`
  - Right: `สุขภาพการเงินของฉัน`

Save as `assets/richmenu.png` (or pass custom path to script).
